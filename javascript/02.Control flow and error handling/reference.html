JavaScript 表达式

表达式是JavaScript中任何能产生值的代码单元。

什么是表达式

表达式是JavaScript中任何能产生值的代码单元。表达式可以是：

  • 字面量（直接量）：如数字、字符串
  • 变量
  • 函数调用
  • 运算符组合

示例

// 数值表达式
5 + 3

// 字符串表达式
"Hello " + "World"

// 变量表达式
let x = 10;
x * 2

// 函数调用表达式
Math.max(10, 20)

JavaScript 算术运算符

算术运算符用于执行数学运算。

基本算术运算符

运算符 描述 示例
+ 加法 5 + 2 = 7
- 减法 5 - 2 = 3
* 乘法 5 * 2 = 10
/ 除法 5 / 2 = 2.5
% 取模（余数） 5 % 2 = 1
** 幂运算 5 ** 2 = 25

示例

let a = 10;
let b = 3;

console.log(a + b); // 13
console.log(a - b); // 7
console.log(a * b); // 30
console.log(a / b); // 3.333...
console.log(a % b); // 1
console.log(a ** b); // 1000

JavaScript 比较运算符

比较运算符用于比较两个值，返回布尔值（true或false）。

比较运算符

运算符 描述 示例
== 等于 5 == "5" // true
=== 严格等于 5 === "5" // false
!= 不等于 5 != "6" // true
!== 严格不等于 5 !== "5" // true
> 大于 5 > 3 // true
< 小于 5 < 3 // false
>= 大于等于 5 >= 5 // true
<= 小于等于 5 <= 3 // false

示例

let x = 5;
let y = "5";
let z = 10;

console.log(x == y); // true
console.log(x === y); // false
console.log(x != z); // true
console.log(x !== y); // true
console.log(x > z); // false
console.log(x < z); // true
console.log(x >= 5); // true
console.log(x <= 4); // false

JavaScript 逻辑运算符

逻辑运算符用于组合或操作布尔值。

逻辑运算符

运算符 描述 示例
&& 逻辑与 true && false // false
|| 逻辑或 true || false // true
! 逻辑非 !true // false

示例

let a = true;
let b = false;
let age = 25;
let hasLicense = true;

console.log(a && b); // false
console.log(a || b); // true
console.log(!a); // false

// 实际应用示例
console.log(age >= 18 && hasLicense); // true
console.log(age < 18 || !hasLicense); // false

JavaScript 条件语句

条件语句用于基于不同条件执行不同的代码块。

if...else 语句

if语句用于在条件为真时执行代码块，else语句用于在条件为假时执行代码块。

if...else 示例

let hour = new Date().getHours();
let greeting;

if (hour < 12) {
  greeting = "上午好！";
} else if (hour < 18) {
  greeting = "下午好！";
} else {
  greeting = "晚上好！";
}

console.log(greeting);

switch 语句

switch语句用于基于不同情况执行不同的代码块。

switch 示例

let day = new Date().getDay();
let dayName;

switch (day) {
  case 0:
    dayName = "星期日";
    break;
  case 1:
    dayName = "星期一";
    break;
  case 2:
    dayName = "星期二";
    break;
  case 3:
    dayName = "星期三";
    break;
  case 4:
    dayName = "星期四";
    break;
  case 5:
    dayName = "星期五";
    break;
  case 6:
    dayName = "星期六";
    break;
  default:
    dayName = "未知";
}

console.log("今天是 " + dayName);

JavaScript 循环语句

循环语句用于重复执行代码块。

for 循环

for循环在已知迭代次数时使用。

for 循环示例

// 打印1到5的数字
for (let i = 1; i <= 5; i++) {
  console.log("当前数字: " + i);
}

// 计算1到10的和
let sum = 0;
for (let i = 1; i <= 10; i++) {
  sum += i;
}
console.log("1到10的和: " + sum);

while 循环

while循环在条件为真时重复执行代码块。

while 循环示例

// 打印1到5的数字
let i = 1;
while (i <= 5) {
  console.log("当前数字: " + i);
  i++;
}

// 猜数字游戏
let number = 7;
let guess = 0;
while (guess !== number) {
  guess = Math.floor(Math.random() * 10) + 1;
  console.log("猜测: " + guess);
}
console.log("猜对了！数字是 " + number);

do...while 循环

do...while循环先执行一次代码块，然后在条件为真时重复执行。

do...while 循环示例

let count = 1;
do {
  console.log("计数: " + count);
  count++;
} while (count <= 5);

// 即使条件一开始就为false，也会至少执行一次
let x = 10;
do {
  console.log("x的值: " + x);
  x++;
} while (x < 5);
