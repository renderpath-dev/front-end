# TypeScript

This directory contains TypeScript type system practice.

## Topics

- Basic types
- `any` and `unknown`
- Function types
- Object types and interfaces
- Type aliases
- Union and intersection types
- Type narrowing
- Generics
- Utility types

## Goal

Use TypeScript to model program behavior clearly, not just add type annotations mechanically.
