# Nuxt Full-stack Boundary Lab

This subproject is a standalone Nuxt lab for Vue Chapter 12. It is intentionally separate from the root Vite learning shell.

## Commands

```bash
npm install
npm run prepare
npm run dev
npm run typecheck
npm run build
npm run generate
npm run preview
```

Development runs on `http://localhost:3030`.
Preview runs on `http://localhost:3031`.

## Scope

- File-based pages
- Layouts
- Route middleware
- Client, server, and universal plugins
- `useFetch`
- `useAsyncData`
- Nitro `server/api`
- Nitro `server/routes`
- Runtime config public/private boundary
- SSR-safe and client-only components
- Hydration mismatch and fixed patterns
- Route rules for prerender and hybrid rendering
- SEO and meta examples

No database, auth provider, Nuxt module, cloud SDK, or deployment provider file is included in this chapter.
