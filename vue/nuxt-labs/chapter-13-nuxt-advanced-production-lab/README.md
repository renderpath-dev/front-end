# Nuxt Advanced Production Lab

This Chapter 13 lab is a separate Nuxt application inside the Vue learning workspace. It demonstrates Nuxt modules, content, image rendering, sealed cookie sessions, protected server routes, local layers, and focused tests.

## Commands

```bash
npm install
npm run prepare
npm run typecheck
npm run test
npm run dev
npm run build
npm run generate
npm run preview
```

Dev server: `http://localhost:3040`

Preview server: `http://localhost:3041`

## Demo users

- `learner@example.com` / `learn-nuxt-13`
- `admin@example.com` / `admin-nuxt-13`

## Environment examples

Copy `.env.example` to `.env` only for local development. Keep `NUXT_SESSION_PASSWORD` private and at least 32 characters.
