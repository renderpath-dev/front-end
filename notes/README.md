# Notes

This directory contains learning notes, summaries, and review materials.

## Suggested Files

```text
notes/
├── javascript.md
├── typescript.md
├── react.md
├── nodejs.md
└── engineering.md
```

## Rule

Notes should explain concepts in your own words and connect them to code.

Good notes should include:

- Concept
- Runtime or type-system mechanism
- Code example
- Common mistake
- Summary
