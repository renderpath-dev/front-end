# Mini Projects

This directory contains small front-end projects and feature demos.

Mini projects are used to connect individual concepts into usable features.

## Suggested Projects

```text
mini-projects/
├── todo-list/
├── form-validator/
├── markdown-preview/
├── weather-card/
└── local-storage-notes/
```

## Rule

A mini project should have its own README explaining:

- What it does
- What concepts it practices
- How to run it
- What can be improved later
